## About

<p align="justify">
Pixel-perfect implementation of a Figma design, built with vanilla HTML, CSS, and JavaScript. Features task management with real-time date display via API integration.
</p>

---

## Features

- Add new tasks to the list
- Update information for existing tasks
- Mark tasks as completed
- Remove tasks from the list
- Hide and show completed tasks with the "Hide/Show Completed Tasks" button.
- Display the current day of the week and date through an integrated API.
